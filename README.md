## Usage
```
ycrawler.py [-db] [-t] [-n] [-l]
```
## Description
Download news from news.ycombinator.com

## Optional arguments:
```
    -db, --database     System base path
    
    -t,  --timeout      TIMEOUT time for scan
    
    -n,  --news         Path for save news
    
    -l,  --log          Path for logfile
```
